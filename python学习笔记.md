##### 格式化输出

| 格式化字符 | 含义                                                         |
| ---------- | ------------------------------------------------------------ |
| %s         | 字符串                                                       |
| %d         | 有符号十进制整数，%06d表示输出的整数显示位数，不足的地方使用0补全 |
| %f         | 浮点数，%.02f表示小数点后只显示两位                          |
| %%         | 输出%                                                        |
